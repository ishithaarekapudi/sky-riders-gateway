# Sky Riders Gateway

Sky Riders Gateway is a youth-centered aviation platform that connects students to personalized roadmaps, scholarships, mentors, organizations, career pathways, and learning resources.

## Built with

- Next.js 16
- React 19
- TypeScript
- Responsive custom CSS

## Pages

- Home
- Explore Your Path
- Scholarships
- Organizations
- Careers
- Resources

## Run locally

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.
